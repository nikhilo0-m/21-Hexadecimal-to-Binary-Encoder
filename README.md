# 21-Hexadecimal-to-Binary-Encoder-16-to-4-Encoder-
Hexadecimal-to-Binary Encoder in Verilog HDL. The encoder converts one active input among 16 input lines into a 4-bit binary output. The design is developed using combinational logic and verified through simulation with a testbench in Xilinx Vivado. Functional correctness is confirmed using waveform analysis.
